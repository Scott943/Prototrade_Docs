Quickstart

  1. Ensure Python 3.8 or higher is installed (use python3 --version to check)

  2. Use pip install prototrade -U to install the latest version of the package.

  3. Create a new python file (e.g. script.py) and paste in the boilerplate strategy .

  4. Ensure you have an API username and API key from Alpaca <https://alpaca.markets> _ .

  5. Place in your API username and API Key within the initaliser for the StrategyRegistry class.

  6. Use python3 script.py to run the boilerplate code.

* Alpaca offers a paid subscription alongside a free plan. The only market available with the free plan is 'iex'. With the paid subscription, you can change the market parameter to 'sip' to receive quotes across all US markets.

Look at the tutorial below and the Example Strategies section for more examples of how the framework operates.

Creating a Strategy Function

Below are the key components needed for creating a strategy, following the same format as the boilerplate strategy .

  • Import the StrategyRegistry from prototrade.strategy_registry at the top of your python script.

  • Define a function that takes an Exchange object as its first argument. The Exchange parameter is how a strategy can retrieve the latest quotes for subscribed stocks and place orders for symbols.

  • The strategy function can also take as many other arguments as you desire.

  • Subscribe to desired symbols (stocks) using the subscribe member function of the Exchange object.

  • Create a while loop that repeatedly checks whether the exchange is running with the member function is_running

Retrieving Stock data

  • To (un)subscribe to particular symbols, use the subscribe and unsubscribe functions from the Exchange object.

  • Use get_subscribed_quotes to get the latest quotes for all the stocks that are subscribed to.

  • To access historical data use the Historical property in the Exchange class.

Placing Orders

  • To submit orders, use the create_order member function.

  • Orders can be cancelled with the cancel_order member function.

  • See the Exchange for more available functions.

Registering Strategies for Execution

  • Ensure the StrategyRegistry class has been imported

  • In the script’s main() function, create an instance of the StrategyRegistry class

  • Use the member function, register_strategy to register a function/strategy to the framework

  • When all strategies are registered, use the member function, run_strategies to start the simulated execution of the strategies.

Creating Graphs

As an example of plotting graphs, we can use pandas and matplotlib:

import pandas as pd
import matplotlib.pyplot as plt

To graph data for a particular stock, use the Historical property in the the Exchange class.

The following execerpt was taken from the plotting stock prices example strategy.

aapl_price_bars = exchange.historical.get_bars("AAPL", "1minute", "2021-01-13", "2021-01-13").df
# Reformat data (drop multiindex, rename columns, reset index)
aapl_price_bars.columns = aapl_price_bars.columns.to_flat_index()
aapl_price_bars.reset_index(inplace=True)

# Plot stock price data
plot = aapl_price_bars.plot(x="timestamp", y="close")
plot.set_xlabel("Date")
plot.set_ylabel("Apple Close Price ($)")
plt.savefig("aapl_bars")

To graph profit and loss, you can use the get_pnl_over_time method. N.B. this function is expensive, so try to not use this every while loop.

pnl_over_time = exchange.get_pnl_over_time() # returns a list of lists.
pnl_df = pd.DataFrame(pnl_over_time, columns = ['timestamp', 'pnl']) # convert to dataframe
plt.plot(pnl_df['timestamp'], pnl_df['pnl'])
plt.xlabel("TimeStamp")
plt.ylabel("Profit / Loss")
plt.gcf().autofmt_xdate()
plt.savefig("pnl_for_strategy")

See plotting pnl for a complete example of creating PnL graphs.

To graph how positions change over time, use the get_positions_over_time method:

pos_over_time = exchange.get_positions_over_time("AAPL") # returns a list of lists.
if pos_over_time:
   pos_df = pd.DataFrame(pos_over_time, columns = ['timestamp', 'symbol', 'position']) # convert to dataframe
   plt.plot(pos_df['timestamp'], pos_df['position'])
   plt.xlabel("TimeStamp")
   plt.ylabel("AAPL Position Amount")
   plt.gcf().autofmt_xdate()
   plt.savefig("positions_for_strategy")

With a bit more effort, we can plot position data for multiple symbols on the same data. This is particularly useful when a strategy holds positions over multiple stocks (see plotting positions multi).

if pos_over_time:
   pos_df = pd.DataFrame(pos_over_time, columns = ['timestamp', 'symbol', 'position']) # convert to dataframe
   fig, ax = plt.subplots()

   for symbol in pos_df['symbol'].unique():
      rows = pos_df[pos_df.symbol==symbol]
      ax.step(rows.timestamp, rows.position,label=symbol)

   ax.set_xlabel("TimeStamp")
   ax.set_ylabel("Position Amount")
   ax.legend(loc='best')
   plt.gcf().autofmt_xdate()
   fig.savefig("positions_for_strategy_multi")
