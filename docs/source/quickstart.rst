
Quickstart
======================================

1. Use ``pip install prototrade -U`` to install the latest version of the package
2. Create a new python file (e.g. ``script.py``) and paste in the :py:class:`minimal boilerplate strategy <example_strategies.minimal_boilerplate>` below.
3. Use ``python3 script.py`` to run the boilerplate code.
