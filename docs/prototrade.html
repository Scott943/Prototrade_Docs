class prototrade.strategy_registry.StrategyRegistry(streamer_name, streamer_username=None, streamer_key=None, exchange_name='iex', save_data_location=None)[source]

Bases: object

The master class used for initialising & running the framework. Handles allocating processes to each of the registered strategies.

Raises

ExchangeNotOpenException – If the exchange is currently closed, this error is raised

register_strategy(strategy_func, *args)[source]

Register a function to be executed. Include any arguments that the strategy function should be provided (any number of arguments is permissible). N.B. When the strategies are started, each strategy function is always provided with an Exchange object as the first parameter. This enables the strategy to interact with the framework.

Parameters

  • strategy_func (function) – A strategy function to register

  • *args (any) – the parameters to pass to the strategy function, optional
